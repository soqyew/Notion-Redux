# Notion

## Описание проекта

Дааное приложение позволяет пользователю зарегестрироваться на странице и оставлять там свои заметки.

## Страницы приложения
* Регистрация
* Вход
* Домашняя страница
* Заметки
  - Редактирование
  - Создание
  - Просмотр


## Технологии

* React
* Zod
* React hook Form
* React Router
* Redux
* json-server
* Tailwind css

## Запуск приложения
1. npm install
2. npm start
3. npm run dev:db
